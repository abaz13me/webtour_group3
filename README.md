# Wonderful Cirebon - Virtual Tour Alun-Alun Kejaksan

Welcome to the **Wonderful Cirebon** Virtual Tour project! This modern, responsive website showcases the beauty and history of Alun-Alun Kejaksan Cirebon through an immersive 360-degree virtual tour and detailed destination pages.

## Features

-   **Immersive 360° Virtual Tour**: Allows users to explore Alun-Alun Kejaksan as if they were there.
-   **Modern Design**: Inspired by "Wonderful Indonesia", featuring a clean, vibrant, and premium aesthetic.
-   **Glassmorphism Effects**: Stylish, transparent elements for a contemporary look.
-   **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices.
-   **Interactive Elements**: Smooth scrolling, hover effects, and entrance animations.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling, variables, flexbox, and grid.
-   **JavaScript**: Navbar interactions and dynamic behavior.
-   **Bootstrap 5.3**: Rapid response grid and components.
-   **Google Fonts**: 'Outfit', 'Inter', and 'Dancing Script' for typography.
-   **Pannellum** (Integrated): For the 360-degree panorama viewer.

## Setup & Deployment

1.  **Local Development**:
    -   Clone or download the repository.
    -   Open `index.html` in your web browser.
    -   To start the 360 tour properly, you may need a local server (e.g., VS Code Live Server) due to browser security policies regarding local file access.

2.  **Deployment**:
    -   Host on GitHub Pages, Netlify, or Vercel.
    -   Simply upload the files and point the domain to `index.html`.
